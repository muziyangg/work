接口说明

仪器厂商 贝克曼生化流水线Remisol

通讯方式 主动上传 仪器类型 生化免疫流水线
医院名称
程序名称 MI.MIFRemisol

仪器上接口标示和接口类型

  1. 接口类型：通过监听上传文件双向
  2. 连接方式：.hpr文件和.OK两个同目录不同文件

仪器接口说明

  • 仪器通过中间件共享到内网电脑，在内网电脑监听共享文件夹进行文件传输达成双向。一般在UpLoad文件读取文件，DownLoad上传双向文件
  • 通过旗标进行报警信息转换，生化项目较多，存储到结果说明造成his可以看到仪器的报警信息，因此本接口将报警信息保存到结果第三列其他结果处。
  • 对于BNP N端脑钠肽前体特殊处理，对于不同年龄不同结果有不同的解释，通过ConvertresKZJG方法进行转换，根据实际情况进行是否屏蔽。
  • 通过PreDeal.DealDemoM,PreDeal上传类，因此在M中可以写好文件名，文件内容后上传双向文件。输出本质是query，输出一条数据上传一个文件。

串口盒子配置

  • 波特率： 9600
  • 数据位：8
  • 停止位： 1
  • 校验位： N

/// 获取扩展结果
ClassMethod ConvertresKZJG(mi, epis, code, res)
{
	s code=$g(code),res=$g(res)
	s OutPut=""

	s Labno=$$GetLabnoByEpis(mi,epis)
	i $l(Labno),$d(^dbo.RPVisitNumberI("IndexVisitNumber",Labno)) d
	.s VisitNumberDR=$o(^dbo.RPVisitNumberI("IndexVisitNumber",Labno,""))
	.s Age=$lg($g(^dbo.RPVisitNumberD(VisitNumberDR)),18)
	s Age=$g(Age)

	i '$l(code),'$l(res) q OutPut
	i code="BNP",0 d		// BNP N端脑钠肽前体
	.i '$l(Age) q
	.i Age<=50 d
	..i res<=450 s OutPut="正常"
	..i res>450,res<=600 s OutPut="急性心衰"
	..i res>600 s OutPut="慢性心衰"
	.i ((Age>50)&&(Age<=75)) d
	..i res<=900 s OutPut="正常"
	..i res>900 s OutPut="急性心衰"
	.i Age>75 d
	..i res<=1800 s OutPut="正常"
	..i res>1800 s OutPut="急性心衰"

	i code="25OHD" d		// 25羟维生素D
	.i res<=20 s OutPut="缺乏"
	.i res>20,res<=30 s OutPut="不足"
	.i res>30,res<=100 s OutPut="充足"
	.i res>100 s OutPut="超上限"
	q OutPut

GetLabnoByEpis(mi,Epis)
	s ret=Epis,mi=$g(mi)
	s date=$zd($h,8)
	i '$l(mi) q Epis
	s WGMachineID=$lg($g(^dbo.BTMIMachineParameterD(mi)),6)
	i '$l(WGMachineID) q Epis
	i $d(^dbo.RPVisitNumberReportI("IndexEpisodeNo",date,WGMachineID,##Class(LIS.Util.Common).IndexData(Epis))) {
		s ReportDR=$o(^dbo.RPVisitNumberReportI("IndexEpisodeNo",date,WGMachineID,##Class(LIS.Util.Common).IndexData(Epis),""),-1)
		S VisitNumberDR=$LG($G(^dbo.RPVisitNumberReportD(ReportDR)),2)
		s ret=$lg(^dbo.RPVisitNumberD(VisitNumberDR),2)
		}
	q ret
}

/// ImageClass:图片类别，如果是监听图片模式该位置放图片名称，自己在保存前提取图片类型
/// FileName:保存在文件服务的相对路径，默认不动
/// FullName:文件全路径，如果是监听图片模式该位置放图片全路径名称，满足有的图片名称无法得到流水和图片类别的情况

{
	//通过OtherPara配置取图的在这里通过epis和ImageClass的图片名字提取出流水号和图片类别
    s mi=$g(mi)
    s epis=$g(epis)
    s ImageClass=$g(ImageClass)

    s Name=epis
    s epis=$p(Name,"_",1)
    s ImageClass=$p(Name,"_",2)
    s ImageClass=$p(ImageClass,".",1)

    s FullName=$g(FullName)

    d ##Class(MI.Common.MIFBase).Trace(mi,epis_":"_ImageClass_":"_FileName,"H<--M")
    s (ReceiveDate, ImageOrder, Caption, DisplayRatio, Height, Width, Sequence)=""
    //加上当前时间，防止浏览器缓存

    s ret=##Class(MI.Common.MachineResult).SaveImage(ReceiveDate, epis, ImageClass, ImageOrder, FileName, mi, Caption, DisplayRatio, Height, Width, Sequence)
    q ret

/// 得到标本信息
/// w ##class(MI.MIFRemisolN).GetLabnoInfo(24,"24092600054")
ClassMethod GetLabnoInfo(miRowId, labno) As %String
{
 
	S miRowId=$G(miRowId),labno=$G(labno)
	K PLIST
	Q:'$L(miRowId)||'$L(labno) 0
	 
	K ^TMP("MIFTESTCODE",$J,miRowId,labno)
	S visitNumberDR=$O(^dbo.RPVisitNumberI("IndexVisitNumber",##Class(LIS.Util.Common).IndexData(labno),""))
	Q:'$L(visitNumberDR) 1
	S visitNumberData=$G(^dbo.RPVisitNumberD(visitNumberDR))
	Q:'$L(visitNumberData) 2
	D ScanOne^MI.MIF000(miRowId,labno)
   	S chl="",chlList="",hcg5d="",chlCnt=0
   	F {
		S chl=$O(^TMP("MIFTESTCODE",$J,miRowId,labno,chl))
		i $l(chl)'=0   
		{
		    S:$E(chl,1)="A" chlCnt=chlCnt+1
		    S:chl="A179" hcg5d=chl
		    Continue:chl="A179"
		    S chlList=chlList_"^^^"_chl_"\"
		}  
		else
		{
			Q:'$L(chl)
		}
		 
	}
	 
	I $L(hcg5d) {
		I (chlCnt'=2)&&(chlCnt'=3)&&(chlCnt'=4) {
			S hcg5d="A178"
		}
		S chlList=chlList_"^^^"_hcg5d_"\"
	}
	S chlList=$E(chlList,1,$L(chlList)-1)
